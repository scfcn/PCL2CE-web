    2.10.3
    稳定版
    2025-02-16
  • 此版本基于主线版本 2.8.13

    • 更新亮点

    • 本体更新终于有国内服务器了！
    • 新增了更新通道机制，你可以选择 Fast Ring 以获取新功能预览，但这些版本可能有些不稳定！
    • 资源包、光影包的管理实现与 Mod 管理统一，现已支持更新等操作

    新功能

    • 允许自动检查启动器更新并静默更新 (PR @tangge233 / #225 )
    • 添加了更新通道机制，目前分为 Fast Ring 和 Slow Ring (PR @Pigeon0v0 / #239 )
    • 添加了国内本体更新和公告服务器 (PR @Pigeon0v0 / #239 ; 感谢 @pysio2007 )
    • 支持创建多个收藏夹，批量下载 Mod，分享收藏夹 (@LanXi387 / #87 ; PR @tangge233 / #249 )
    • 支持来自 MCIM 的资源简介翻译 (PR @JustQiyi / #242 )

    优化

    • 使用 CE 分支帮助库 (@Gongyifan114 / #196 ; PR @wyc-26 / #227 )
    • 给反馈和新功能投票添加了刷新按钮 (@dannisyang53 / #159 ; PR @tangge233 / #229 )
    • 优化深色模式 MyHint 控件效果 (@NatsumiXD / #222 等; PR @Hill23333 / #223 等 )
    • 调整了关于页面的版本说明 (PR @Pigeon0v0 / #239 )
    • 非必要不启用 Java Wrapper (PR @EncVar / #263 )
    • 针对 ARM64 进行了部分优化，Java Wrapper 在 ARM64 架构的设备上强制禁用 (PR @Pigeon0v0 / #260 )

    Bug 修复

    • Authlib-Injector 验证可能失败 (PR @mqingxue / #208 )
    • 自动安装不会提示 Quilt 与 OptiFine 不兼容 (@Gongyifan114 / #216 ; PR @wyc-26 / #224 )
    • 修正了 UA 和 Referer 以避免一些过滤问题 (PR @Silverteal / #148 )
    • 深色模式下部分控件可能不正常闪烁 (@Drumda / #233 ; PR @Hill23333 / #243 )
    • Windows 版本判断逻辑有误 (PR @Pigeon0v0 / #239 )
    • 静默检查更新会意外弹出提示 (PR @Pigeon0v0 / #239 )
    • 查看版本包含的资源包时可能崩溃 (@FFDVDGD / #245 ; PR @tangge233 / #250 )
    • 部分情况下意外使用了官方版本 PCL 的日志文件 (PR @youzi-3222 / #268 )
    • 意外地和官方版本 PCL 共用了 WebP 支持库文件 (PR @tangge233 / #274 )

    技术性更改

    • 减少了一些编译警告 (PR @tangge233 / #231 )
    • 支持 ARM64 自动编译 (PR @Deep-Dark-Forest @wyc-26 等 / #259 )
    • 调整了仓库开发流程 (by @Deep-Dark-Forest )
